Interfaces y códigos para diseño Web

March 23, 2023 Lab 2

Hoy vimos interfaces y códigos para diseño web.

Iniciamos con conceptos sobre el HTML, el cual consiste en una serie de elementos que usaremos para encerrar distintas partes de un contenido. Para que entendamos mejorar a que nos referimos con elemento, pasaremos a ver uno de ellos, que de seguro ya lo hemos visto pero no sabíamos que se trataba de eso.

Recordemos que un elemento cuenta con partes, iniciando por una etiqueta, atributo "valor" contenido y un slash (/) etiqueta para cerrar el elemento.

Veamos algunas etiquetas que nos ayudarán mucho para elaborar nuestra web en formato HTML.

  • DOCTYPE: sirve para avisar el tipo de documento.
  • Head: va al inicio de nuestro código para dar pie al encabezado.
  • Title: nos ayuda a establecer títulos que aparecerán en las pestañas.
  • Body: es el cuerpo en el que va toda la información sea; videos, fotos, texto, gift, emoticones, audios reproducibles, juegos y demás.
  • Imágenes: estas se incrustan a la página en la posición en que aparece. Se logra a través del atributo src (source), el cual contiene el path (ruta o ubicación) de tu archivo imagen.
  • Encabezados: no solo existe el body, HTML posee seis niveles de encabezados, h1 - h6, aunque probablemente solo lleguemos al 3-4 como mucho. No es recomendable tener muchos, porque malogras la visualización del lector.
  • Listas: Pueden ser ordenadas (OL) o desordenadas (UL), según lo requiera tu publicación.
  • Vínculos: Son de lo más importantes ya que la web nos permite conectar con otros materiales, redes, archivos. Para realizarlo, debemos usar la etiqueta "a", la cual es la abreviatura de ancla.
  • Párrafos: Para ello se utiliza la etiqueta p y cerramos el párrafo con /p.
  • Atributos: en el caso de "class" permite dar el nombre identificativo, que se puede utilizar luego para apuntarle al elemento, como información de estilo y demás cosas.

En realidad, el manejo de HTML está siendo de fácil entendimiento y me está gustando cada vez más. ¡Hasta la próxima!
