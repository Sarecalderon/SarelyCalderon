Bitácora 12

Programación de Arduino

April 28, 2023 Lab 2

En esta clase vimos sobre la programación que se necesita para realizar proyectos en arduino, la representación de diagrama de flujo en arduino y la configuración de pines arduino.

Algunos sentencias de configuraciones de pines fueron:

  • pinMode (pin, mode): usado para asignar el modo de un pin. mode puede ser INPUT o OUTPUT.
  • digitalWrite (pin, value): una vez un pin es asignado como OUTPUT se puede asignar a HIGH o LOW.
  • int digitalRead (pin): una vez un pin es asignado como INPUT se puede usar para devolver HIGH o LOW.
  • analogWrite (pin, value): algunos de lo pines sorpotan pulsos modulados. Pone a un pin en ON o OFF muy rápido.
  • int analogRead (pin): puede leer un voltaje. Devuelve el valor entre 0 y 1024.

Sentencias if else-elseif: Sirven para que el programa haga una cosa u otra dependiendo de cada situación (sí-no).

El loop en un diagrama de flujo funciona de manera que se ejecuta durante un número de repeticiones especificado o hasta se cumplen las condiciones específicas.

Operadores relacionales

  • == igual a
  • != no igual a
  • > mayor que
  • >= mayor o igual que
