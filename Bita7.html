Bitácora 7

Propiedades y argumentos de CSS

April 13, 2023 Lab 2

Esta semana empezamos con algunas propiedades y argumentos que nos ayudaran a mejorar la forma estetica de nuestra web.

Estos son algunos de lo términos que nos aportaran orden y estética.

  • Id= Este argumento brinda una identificación a solo un elemento.
  • Class: Brinda identificación a varios elementos que se encuentren agrupados y se puede realizar el llamado del valor, desde CSS.

Además, para gregarle color a algunos elementos de la web utilizamos "color", esta propiedad permite especificar el color de texto en 3 formatos.

  • Valores RGB
  • Códigos HEX
  • Nombre de color

Asimismo, hay veces en las que queremos jugar con la opacidad para lograr ciertos estilos, es por eso que lo podemos hacer a través de:

  • Opacity: 0.5;) Con ello estaríamos graduando el color a un 50%, si queremos que sea color sólido sería 1.0 y para cerrar siempre es con ";"

A su vez, podemos editar las variables del color a través de Colores HSL.

  • H hue: Se refiere al tono.
  • S saturation:Se refiere a la intensidad de color.
  • L lightness:Se refiere a la luminosidad que se le quiere dar.

Por otro lado, para poder definir la tipografía que quieres usar en tu web es recomendable que sea una que usualmente viene instalada con las PCs como: Arial, Verdana, entre otras. Ya que si la persona abre tu web y no tiene instalada tu fuente, no podrá ver absolutamente nada del contenido.

CSS: Se trabaja con el Font Family y Font Face.

Con ello también vimos algunas formas de transformación de texto como:

  • Uppercase: Se refiere al texto en mayúsulas.
  • Lowercase:Se refiere al texto en minúscula.
  • Capitalize:Se refiere a que el texto inicia con la primera letra mayúscula.

Por último, vimos las alineaciones de texto (left, center, justify, right) y los espaciados entre caracteres y palabras.
